Affordance

É o potencial de um objeto ou interface de sugerir suas funções ao usuário. Por exemplo, um botão em uma página web tem uma forma e cor que indicam que ele pode ser clicado. O conceito vem da psicologia e é aplicado ao design para melhorar a interação do usuário.[1]

Semiótica

Estuda os signos e seus significados dentro de uma cultura. No design, a semiótica é usada para entender como os elementos visuais (como ícones, cores, formas) transmitem mensagens e orientam o comportamento do usuário.[2]

Usabilidade

Refere-se à facilidade com que uma pessoa pode usar uma interface para alcançar seus objetivos. Em design de interação, usabilidade envolve eficiência, aprendizado e satisfação do usuário ao utilizar um sistema ou produto.[3]

CSS

CSS (Cascading Style Sheets) é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Ela controla o layout, cores, fontes e a aparência geral das páginas web, separando a estrutura do conteúdo da estética da página. Esses conceitos são fundamentais no design e desenvolvimento de interfaces, pois buscam criar uma experiência mais intuitiva e eficiente para os usuários.[4]
